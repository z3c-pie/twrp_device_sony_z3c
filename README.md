# TWRP for Sony Xperia Z2

Minimal tree to build TWRP in android-9.0

## Compile

export ALLOW_MISSING_DEPENDENCIES=true

. build/envsetup.sh && lunch omni_sirius-eng

mka adbd recoveryimage
